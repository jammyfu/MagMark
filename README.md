## 🏷️ 为什么叫 MagMark？

**MagMark** 是由两个核心概念组合而成的：

*   **Mag** (取自 **Magazine**)：代表我们的目标——打破 Markdown 预览“简陋”的印象，赋予文字具有现代杂志感的排版美学。
*   **Mark** (取自 **Markdown**)：代表我们的初心——坚持轻量级、纯文本的创作体验，让您专注于内容。

**MagMark = 像写 Markdown 一样简单，像做杂志一样精美。**

## 🚀 1.4 核心升级：彻底的视觉突破

### 1. 🎞️ 彻底抛弃 PDF 中间层
不再依赖“PDF 存图 + PDF 转图片”的冗长链路。1.4 版本直接采用 **高精度 Canvas + SVG 混合采样流水线**：
- **3x 超采样**：输出 600DPI 级别的超清 PNG。
- **字体嵌入**：完美保留全部字体细节，动态抗锯齿处理。
- **所见即所得**：预览效果与最终导出的像素级完全一致。

### 2. 📖 强制两端齐行 + 自动连字符
- **两端齐行 (Text Justify)**：正文强制执行 `text-align: justify`，实现严整的杂志边框感。
- **自动连字符 (Hyphenation)**：原生开启自动连字符处理。
- **CJK 深度优化**：完美解决中文排版中常见的“一边长一边短”问题，阅读节奏感拉满。

### 3. 🖱️ 块级点击浮动微调 (Exclusive!) ✨
**像编辑纸媒一样编辑预览！**
- **点击即编辑**：在预览区直接点击任何文字块（标题、段落、代码块），立即激活 **BLOCK EDIT** 模式。
- **浮动工具栏**：弹出的玻璃拟态工具栏支持针对该区块的**字号、行高、字间距**进行像素级微调。
- **即时重选**：精细化控制每一个段落的视觉重心。

### 4. 📄 智能分页控制 1.4
- **手动分页开关**：`---` 分割线默认不再冲突，仅在开启“手动分页”时作为绝对分页符生效。
- **单页独立样式**：支持为每一页单独设置字号、行距、字间距，轻松制作杂志封面或特殊导读页。
- **动态重算**：调整参数后，分页引擎在 <100ms 内完成重排，零延迟。

## ✨ 核心特性

### 🎨 极致字体排版
- **杂志级排版标准**：精准控制行高、段间距和层级关系。
- **CJK 智能优化**：中英文/数字间自动增加微小间隔，标点偏移预防。
- **标点悬挂 (Hanging Punctuation)**：实现光学边缘对齐，视觉重心更平稳。

### 🖼️ 代码块多维优化
- **长代码自动换行**：代码块支持 `pre-wrap` 和 `break-all`，确保在任何排版宽度下都不会溢出。
- **代码导出优化**：高精度渲染代码高亮细节。

## 🚀 快速开始

### 安装

```bash
npm install magmark-1.4
```

### 基础用法

```typescript
import { MagMark } from 'magmark-1.4';

const magmark = new MagMark({
  viewMode: 'multi', // 分页模式
  autoHyphenation: true,
  precision: 3 // 3倍精度导出
});

// 将 Markdown 转换为杂志级 HTML
await magmark.render('# Hello MagMark 1.4');
```

## 🛠️ 本地开发与调试

### 启动开发环境
```bash
npm install
npm run dev
```
访问 `http://localhost:5173/` 开启排版之旅。

### 关键操作流程
1. **内容录入**：在左侧编辑器输入 Markdown 内容。
2. **全局调整**：通过右侧控制栏设置文档的基础字号、行距。
3. **精细排版**：
   - 开启 **“手动分页”** 进行版面分块。
   - 点击具体文字块，使用 **“浮动工具栏”** 进行独立微调。
4. **一键导出**：点击 **“导出 PNG”** 直接获取 3x 超清渲染图。
