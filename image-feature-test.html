图文混排布局 CSS CLASSES

mm-center（居中）

居中图片，下方文字继续排列，不会环绕图片。适合独立展示的插图。

mm-float-left（浮左图文混排）

文字环绕图片左侧排列，实现杂志风格的图文混排效果。文字自然填充图片右侧空白区域，保持良好的视觉节奏。

mm-float-right（浮右图文混排）

文字环绕图片右侧排列，图片悬浮在段落右侧。适合需要图片与正文对话的排版场景。

mm-full（全宽）

全宽图片铺满容器，适合题图、横幅、分隔图等场景。

Markdown 语法扩展 EXTENDED SYNTAX

标准图片（居中）:
  ![说明文字](image.jpg)

浮左 40% 宽（图文混排）:
  ![说明文字](image.jpg){.float-left width=40%}

浮右 35% 宽:
  ![说明文字](image.jpg){.float-right width=35%}

全宽:
  ![说明文字](image.jpg){.full}

居中 60% 宽:
  ![说明文字](image.jpg){.center width=60%}

// 图片面板插入后生成的 Markdown 示例：
  ![美丽的日落](data:image/png;base64,...){.float-left width=45%}
