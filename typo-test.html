MagMark 排版引擎

将您的 Markdown 转换为具备专业字体排版、智能分页和高精度导出的出版级文档。MagMark 彻底革新了导出流水线，带来了媲美 VOGUE 等高端纸媒的视觉体验。

英文单词保持完整：This is a professional typography engine for Chinese magazines, with overflow-wrap:break-word set correctly — words like "professional" stay intact.

混合排版改善：Hello World 与中文混排时，Han.css 自动在 CJK 与拉丁字符之间插入合适间距，"Hello" 和 "World" 作为完整词组不再被强制折断。

像写 Markdown 一样简单，像做杂志一样精美。— MagMark

Paged.js CSS @page 规则（点击"🖨 打印预览"在新窗口中生效）
/* A4 页面边距 + 页脚居中页码 */
@page {
  size: A4;
  margin: 22mm 18mm 28mm 18mm;
  @bottom-center {
    content: counter(page) " / " counter(pages);
    font-size: 8pt; color: #aaa;
  }
}
@page :left { margin-left: 24mm; margin-right: 14mm; }
@page :right { margin-left: 14mm; margin-right: 24mm; }
@page :first { margin-top: 30mm; } /* 首页留白更多 */

/* Vivliostyle 孤行/寡行控制 */
.magmark p { orphans: 3; widows: 3; }
.magmark h1, h2, h3 { break-after: avoid; } /* 标题后不分页 */
.magmark pre, table { break-inside: avoid; } /* 代码块不被分页打断 */
